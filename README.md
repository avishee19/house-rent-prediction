# House Rent Prediction System

## Description
This project predicts house rent category (Low, Medium, High) using Machine Learning.

## Features
- User inputs property details
- Predicts rent category
- Random Forest Classifier

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML/CSS

## How to Run
1. cd backend
2. pip install -r requirements.txt
3. python app.py
4. Open http://127.0.0.1:5000
